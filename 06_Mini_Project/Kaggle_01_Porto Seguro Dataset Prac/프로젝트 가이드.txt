2. 분류 문제
제목 : Santander Customer Satisfaction
설명 : 산탄데르 은행 데이터세트를 기반으로 고객 만족 여부를 예측
링크 : https://www.kaggle.com/c/santander-customer-satisfaction
문제 유형 : 분류

제목 : Porto Seguro’s Safe Driver Prediction
설명 : 포르토 세구로 보험사에서 제공하는 고객 데이터를 활용해 운전자가 보험을 청구할 확률 예측
링크 : https://www.kaggle.com/c/porto-seguro-safe-driver-prediction
문제 유형 : 분류

1. 위에 설명된 4가지 문제 중 "한가지"를 선택하여

2. 머신러닝 예측 모델을 만들되 아래의 내용이 포함되어야 함
  1) 탐색적 데이터 분석 과정 : 분석 내용 노트북에 문서화
  2) 베이스 라인 모델 : 최초 적용된 데이터 특성과 모델 및 그 때의 성능 기록
  3) 성능 개선 과정 : 데이터 가공을 통한 성능 개선, 모델을 통한 성능 개선 등
  -> 하나의 노트북에 모두 넣어도 되고, 나누어서 작성해도 됨
